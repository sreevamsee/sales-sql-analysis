# Sales & Customer Analysis using MySQL

## Project Description
This project performs exploratory data analysis on a sales database using MySQL.
It analyzes customer spending, product performance, and city-wise revenue using SQL.

## Tables Used
- customers
- products
- orders

## Concepts Used
- JOINs
- GROUP BY
- Aggregate Functions
- Subqueries
- Window Functions (RANK)

## Analysis Performed
- Total spending per customer
- City-wise revenue
- Top selling product
- Customers who never ordered
- 2nd highest spending customer
- Ranking customers by spending

## Key Insights
- Hyderabad generates highest revenue
- Laptop contributes highest revenue
- Ravi is the top spending customer
- Accessories sell more in quantity, Electronics generate more value
